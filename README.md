# CalcStats

Простая консольная утилита на C#, которая принимает список чисел и выводит их сумму, среднее арифметическое, минимальное и максимальное значение.

## 📥 Установка

1.  Убедитесь, что на вашем компьютере установлен **.NET 8 SDK** или новее. Проверить это можно командой:
    dotnet --version
2. Скачайте ZIP-архив со страницы мого репозитория с помощью кнопки "Code" → выберите "Download ZIP"
3. Файлы архива распакуйте в папку, затем откройте командную строку(Win + R - напишите cmd)
4. Через cd обратитесь к папке например cd C:Users/Admin/MyPapka
5. После этого можно пользоваться утилитой! Напишите dotnet run -- числа, например dotnet run -- 5 10 15

## Пример вывода

Числа: 5, 10, 15
Сумма: 30
Среднее: 10.00
Минимальное: 5
Максимальное: 15

## Ошибки, которые могут возникнуть
Если вы не введёте числа, например, просто напишите dotnet run, программа не упадёт, она выдаст подсказку. Подсказка выглядит примерно так: 
Вы не указали числа! Пожалуйста, укажите их!
Указать нужно к примеру таким образом: dotnet run -- 5 15 10
